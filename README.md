# Vortex Programming Language

An open source programming language created and maintained by the community members on the [Vortex chatting site](https://vtx.chat)

Curerntly the programming language is only suited for arithmetic with equations passed in with a text file. We are working on implementing syntax and a transpiler that takes in the source code for the vortex programming language and turning it 
into code for Javascript in order to be ran on a website.

## How to run

make sure you have node installed as it is needed for the main.js file to run

```
node [path to main.js file] [argument for the txt file to be parsed]
```

e.g

```
node main.js test.txt
```

the text file should contain equations excluding parentheses
